# Installation Instructions

## Prerequisites
- Python >= 3.12.0
- (Optional) NVIDIA drivers, CUDA, cuDNN for Linux GPU support

## Install from Source
```sh
git clone https://github.com/exo-explore/exo.git
cd exo
pip install -e .
```

## (Optional) Use venv
```sh
python3 -m venv venv
source venv/bin/activate
pip install -e .
```
