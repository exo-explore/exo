## About this wiki

This wiki was generated by [wiki-build](https://github.com/…) — a structured codebase documentation skill inspired by Cognition's [DeepWiki](https://deepwiki.com). Every factual claim cites a specific `path:line-range` in the source so readers can verify and drill down.

**Design principles:**

1. **Source-cited** — no claim without a line reference
2. **Hierarchical** — 3 tiers (architecture / components / workflows), never a flat sprawl
3. **Incremental** — run `wiki-update` after code changes to refresh only affected pages
4. **Not a replacement for inline docs** — complements `docs/architecture.md`, `README.md`, and `CLAUDE.md`

**Known limitations:**

- Pages are generated by LLMs. Always verify load-bearing claims against the cited source before acting on them.
- Discrepancies found during generation (e.g., the 5 vs 6 topics count in `docs/architecture.md:53` vs `src/exo/routing/topics.py:40-51`) are surfaced in the affected pages rather than silently reconciled.
- Forward cross-links to future pages may exist before those pages are built.

To refresh this wiki after code changes: `/wiki-update` (diffs against `.wiki-meta.json.last_indexed_sha` and regenerates only affected pages).
